# 5. 技术规格


| BIO机械爪G2（AB1100） |                 |
| ---------------- | --------------- |
| 额定电源电压           | 24V DC          |
| 最大电源电压           | 28V DC          |
| 静态功耗（最低功耗）       | 0.72W           |
| 峰值电流             | 1.5A            |
| 质量               | 725g            |
| 最大加持力度           | 25N             |
| 行程范围             | 71-150mm（可更换手指） |
| 通信方式             | RS-485          |
| 通信协议             | Modbus RTU      |
| 可编程参数            | 速度(0-4500)、位置(71-150)、夹持力(0-100百分比）       |
| 状态               | 工作状态、电源状态       |
| 反馈               | 抓取检测、滑落检测       |
