# 5.get\_app\_xml\_data
获取Blockly项目文件信息:
获取Blockly文件xml数据信息。
### Message:  
```json
{
    "cmd": "get_app_xml_data",
    "data": {
        "userId": "test",
        "version": "xarm6",
        "appName": "/test (2)"
    },
    "id": "1"
}
```
### Request：  
* cmd
  * string，必填，请求接口
* date
  * Application/json，必填，请求数据
* id
  * string，必填，请求序列号
