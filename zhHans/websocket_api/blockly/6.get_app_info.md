# 6.get\_app\_info
获取Blockly项目目录信息

### Message:  
```json
{
    "cmd": "get_app_info",
    "data": {
        "userId": "test",
        "version": "lite6"
    },
    "id": "1"
}
```
\
### Request：  
* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号
