# 7.app\_save\_info
创建删除Blockly文件夹或文件时调用

### Message:  
```json
{
    "cmd": "app_save_info",
    "data": {
        "userId": "test",
        "config": {
            "lastAccessTime": 1688725164.49637,
            "deviceType": "lite6",
            "name": "myapp",
            "expendKeys": [
                "/[D]USER_APPS",
                "/[D]/sample",
                "/[D]test001"
            ],
            "selectFilePath": "/uu12"
        },
        "children":[
            {
                "type": "dir",
                "uuid": "/[D]USER_APPS",
                "children": [
                {
                    "type": "file",
                    "name": "uu12",
                    "children": [],
                    "uuid": "/uu12"
                }
                ],
                "name": "USER_APPS"
            },
            {
                "type": "dir",
                "name": "sample",
                "children": [],
                "uuid": "/[D]/sample"
            },
            {
                "type": "dir",
                "name": "test001",
                "children": [],
                "uuid": "/[D]test001"
            }
        ]
    },
    "id": "1"
}
```

### Request：  
* cmd
  * string，必填，请求接口
* date
  * Application/json，必填，请求数据
* id
  * string，必填，请求序列号

### Response:  
```
code=0->成功；
code!=0->失败,-为-32是写文件内容时错误
```

#### 请求参数
无
