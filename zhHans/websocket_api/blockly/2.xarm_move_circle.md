# 2.xarm\_move\_circle
圆弧运动接口:
机械臂运行圆弧运动接口，可实现Blockly中圆弧运动模块点击底下‘移动’按钮的运动。
### Message:  
```json
{
    "cmd": "xarm_move_circle",
    "data": {
        "pose1": [250,50,200,0,0,0],
        "pose2": [250,-50,200,0,0,0],
        "percent": 100, 
        "module": true
    },
    "id": "1"
}
```
\
### Request：  
**请求类型：Application/json**
* cmd
  * string，必填，请求接口
* date
  * json，必填，请求数据
* id
  * string，必填，请求序列号

### Response:  
```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```

无
