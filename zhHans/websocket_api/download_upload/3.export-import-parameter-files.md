#  加载导入机械臂参数文件



### 1. 概述

加载导入机械臂参数文件



### 2. Request & Response

**Request**

| **data** | **数据类型** | **必须** | **描述**                                    | **示例值**        |
| -------- | ------------ |--------| ------------------------------------------- | ----------------- |
| path     | string       | 否      | 上传到系统的路径(test/xarm6手臂类型/config) | test/xarm6/config |
| file     | string       | 否      | 上传文件的二进制数据                        | -                 |

**Response** 

```
result:
    'ok' -> 成功
    'Invalid Args'无效的参数 -> 失败
success:
    1 -> 保存文件数据失败
    0 -> 上传成功
```
