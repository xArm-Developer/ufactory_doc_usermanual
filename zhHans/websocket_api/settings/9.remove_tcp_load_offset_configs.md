# 9.remove\_tcp\_load\_offset\_configs



 



删除TCP负载偏移接口：



### Message:  



```
删除负载与偏移
{
    "cmd": "remove_tcp_load_offset_configs",
    "data": {
        "tcp_offset_id":"2de2381ed7141587911361387775",
        "tcp_load_id":"2de2381ed7141587911361387775"
    },
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号

\





### Response:     



```
code=0->成功；
code!=0->失败。
```



\

无
