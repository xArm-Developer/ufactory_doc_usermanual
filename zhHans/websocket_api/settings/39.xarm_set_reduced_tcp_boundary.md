# 39.xarm\_set\_reduced\_tcp\_boundary



 



设置缩减模式的安全边界接口:

设置缩减模式的安全边界范围。



### Message:  



```yaml
{
    "cmd": "xarm_set_reduced_tcp_boundary",
    "data": {
        'boundary': [9999, -9999, 9999, -9999, 9999, -9999],
        'save': true
    },
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号

\





### Response:     



```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```



\





#### 请求参数



无
