# 13.xarm\_get\_reduced\_states



 



获取缩减模式状态接口:

获取缩减模式状态的功能。



### Message:  



```
#获取缩减模式
{
    "cmd": "xarm_get_reduced_states",
    "data": {},
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号



\





### Response:     



```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```



\





#### 请求参数



无
