# 37.xarm\_set\_reduced\_joint\_range



 



设置缩减模式的关节运动范围接口:

设置缩减模式的关节运动范围。



### Message:  



```
{
    "cmd": "xarm_set_reduced_joint_range",
    "data": {
        'save': true, 
        'boundary': [-360, 360, -117.97, 120, -225, 11, -360, 360, -97, 180, -360, 360, 0, 0]
    },
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号

\





### Response:     



```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```



\





#### 请求参数



无
