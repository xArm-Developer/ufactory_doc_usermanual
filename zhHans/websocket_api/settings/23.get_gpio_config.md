# 23.get\_gpio\_config



 



获取控制器数字IO触发的相应操作接口:

获取控制器输入输出IO拉高触发的相应操作指令。



### Message:  



```json
{
    "cmd": "get_gpio_config",
    "data": {},
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号



\





### Response:     



```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```



\





#### 请求参数



无
