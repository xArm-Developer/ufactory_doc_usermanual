# 1.xarm\_set\_effector\_modbus\_rtu\_cmd



192.168.1.95:18333/ws



发送modbus指令:

发送控制盒、机械臂modbus指令。



### Message:  



```
# 使能BIO机械爪
{
    "cmd": "xarm_set_effector_modbus_rtu_cmd",
    "data": {
        "isLoop": false,
        "mdb_info": [{'checked': true, 'note': {'cn': '', 'en': ''}, 'delay': '', 'cmd': "08 10 01 00 00 01 02 00 01"}],
        "end_fmv": "2.5.9",
        "baudrate": 2000000,
        "is_stop": false,
        "host_id": 11
    },
    "id": "1"
}
```

* id
