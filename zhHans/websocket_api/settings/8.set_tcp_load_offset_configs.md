# set\_tcp\_load\_offset\_configs



192.168.1.99:18333/ws



设置TCP负载偏移接口：



### Message:  



```
设置负载与偏移
{
    "cmd": "set_tcp_load_offset_configs",
    "data": {
        "tcp_offset": {
            "disableEdit": false, 
            "id": "2de234a3ec34972658484198871", 
            "name": "69699", 
            "values": [20, 0, 0, 0, 0, 0]
        },
        "tcp_load": {
            "disableEdit": false, 
            "id": "2de234a3ec34972658484198871", 
            "name": "69699", 
            "values": [0, 0, 0, 0]
        },
        "default_id": "2de234a3ec34972658484198871"
    },
    "id": "1"
}
```



\





### Request：    



**请求类型：Application/json**



* cmd

  * string，必填，请求接口

* date

  * json，必填，请求数据

* id

  * string，必填，请求序列号
