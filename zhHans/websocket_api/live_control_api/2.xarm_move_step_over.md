# 2.xarm\_move\_step\_over
停止移动机械臂:
实现按钮松开停止机械臂移动功能，需配合xarm\_move\_step接口一起使用。
### Message:   
```json
{
    "cmd": "xarm_move_step_over",
    "id":"1"
}
```
\
### Request：  
* cmd
  * string，必填，请求接口
* id
  * string，必填，请求序列号

\

### Response:  
```
code=0->成功；
code!=0->失败，详情参考xArm_Python_SDK/xarm_api_code.md文档。
```
\
#### 请求参数
无
