# 7.xarm\_set\_speed
\
设置速度接口:
设置实时控制界面机械臂速度百分比。
### Message:   
  ```
#设置速度百分比为100%
{
    "cmd": "xarm_set_speed",
    "data": {
        "percent": 1
    },
    "id":1
}
```
\
### Request：  
* cmd
  * string，必填，请求接口
* data
  * Application/json，必填，请求数据
* id
  * string，必填，请求序列号

\

### Response:  
```
code=0->成功; code!=0->失败
```
\
#### 请求参数
无
