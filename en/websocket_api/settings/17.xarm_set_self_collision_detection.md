# xarm\_set\_self\_collision\_detection
### 1. Introduction
Enable/Disable 'Self-collision Detection'.
### 2. Request & Response

```json
{
    "cmd": "xarm_set_self_collision_detection",
    "data": {
        'on': True,    
        'save': True,

    "id": "1"
}
```
