# xarm\_set\_bio\_gripper\_enable

### 1. Introduction

Enable bio gripper.

Button: \[Live Control] - \[End Effector] - \[Bio Gripper]- Enable.

```json
{
    "cmd": "xarm_set_bio_gripper_enable",
    "data": {
        "enable": true

    "id": "1"
}
```
