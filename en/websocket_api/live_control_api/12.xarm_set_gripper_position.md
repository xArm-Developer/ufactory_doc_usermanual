# xarm\_set\_gripper\_position

### 1. Introduction

Button: \[Live Control] - \[End Effector] - \[xArm Gripper]- position.

### 2. Request & Response


**Message**
```json
{
    "cmd": "xarm_set_gripper_position",
    "data": {
        "enable": true,
        "pos": "400",
        "wait": false,
        "speed": 3000
    },
    "id": "1"
}
```


**Request**
